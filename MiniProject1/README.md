# MiniProject

This repository was created for the MiniProject.

## Dataset
To test the project, the [Hymenoptera dataset](https://download.pytorch.org/tutorial/hymenoptera_data.zip) needs to be downloaded.

## Scripts

- **`/scripts/train.py`**  
  Trains a model using the parameters you provide.

- **`/scripts/predict.py`**  
  Performs classification on a given input using a trained model of your choice.

- **`/scripts/plot_results.py`**  
  Generates the plots used in the project report.

- **`/scripts/device_test.py`**  
  Compares CPU and GPU performance.

  ## Note
  The trained model weights are not included in this repository due to their large file size.
