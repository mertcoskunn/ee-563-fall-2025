# FinalProject

This repository was created for the FinalProject.

## Installation

- **`Shape-E`**  
 Follow this [link](https://github.com/openai/shap-e) to install Shap-E model. 

- **`Hugging Face`**
```bash
pip install transformers
```

- **`Blender`**
  You can download Blender using this [link](https://www.blender.org/download/). 
  
## Note
Open the `main.py` file and update the `BLENDER_PATH` variable with the full path to your own `blender.exe`.

Example:
```python
BLENDER_PATH = "C:/Program Files/Blender Foundation/Blender 3.6/blender.exe"
```
## Usage
IMG_PATH= input image
```bash
python main.py IMG_PATH
```
